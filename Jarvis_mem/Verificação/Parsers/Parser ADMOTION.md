# Parser ADMOTION

**Arquivo:** `app/verification/parsers/parser_admotion.py`

## Características

- Formato: CM360 (Campaign Manager 360)
- Veículo: **Site** (coluna "Site (CM360)")
- Viewability: colunas Active View
- URL: coluna "URL Veiculada"

> [!info] Mesmo formato que AHEAD
> ADMOTION e AHEAD usam o mesmo formato CM360. O parser AHEAD (`parser_ahead.py`) é essencialmente o mesmo código.
